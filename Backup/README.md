# P2P Chat

A peer-to-peer chat application for Android.

## Features
- P2P communication
- QR code scanning for connection
- Secure and private

## Building
The APK is automatically built and released via GitHub Actions when a new tag is pushed.

## Releases
Check the [Releases](https://github.com/jnetai-clawbot/p2p-chat/releases) page for the latest `p2p-chat.apk`.
